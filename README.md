# 🎮 Hangman Game - CodeAlpha Internship

## 📌 Project Overview

This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player has a limited number of incorrect attempts before the game ends.

---

## 🚀 Features

- Random word selection
- Letter-by-letter guessing
- Maximum 6 incorrect attempts
- Displays guessed letters
- Win/Loss detection
- User-friendly console interface

---

## 🛠️ Technologies Used

- Python 3
- Random Module
